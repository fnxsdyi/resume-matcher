```bash
cd D:\MyMimoProject\resume-matcher

```

### 运行应用
```bash
# 方式1：使用云端 API（推荐，无需安装 Ollama）
streamlit run app.py
# 然后在侧边栏选择 OpenAI/Together AI/API2D 并输入 API Key

# 方式2：使用本地 Ollama
# 1. 安装 Ollama: https://ollama.ai
# 2. 运行: ollama serve
# 3. 拉取模型: ollama pull llama3
# 4. 运行: streamlit run app.py
```
